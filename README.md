# LeetCode 题解集合

本仓库整理常见题型与经典题目，按目录归类，提供可直接在 LeetCode 使用的 C++ 题解。

## 目录结构
- 题型总览与题解汇总见 [题型目录与题解.md](题型目录与题解.md)
- 题解按题型目录存放，例如 [数组](数组) / [字符串](字符串) / [动态规划](动态规划) / [排序算法](排序算法)

## 使用方式
- 单文件 C++ 编译运行：
  - `g++ 题目.cpp -o 题目 && ./题目`
- Java 单文件编译运行：
  - `javac 题目.java && java 题目`

## 贡献建议
- 新增题解按题型目录归类，文件名使用题目中文名加后缀
- 代码优先简洁、可读，保持与现有风格一致
