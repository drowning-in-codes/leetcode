```
1. 打开 容器和方法示例/README.md
2. 在"按场景选择容器"中找到你的需求

4. 编译并运行 .cpp 或 .java 示例

```

### 方案 3：解决 Bug（容器相关问题）
```
1. 打开 最佳实践与常见陷阱.md
2. 在"常见陷阱"部分查找症状（如"迭代删除异常"）
3. 复制"正确做法"代码
4. 检查自己的代码是否有相同问题
```

### 方案 4：性能优化（TLE 需要快）
```
1. 打开 最佳实践与常见陷阱.md
2. 查看"性能对比"表，确认是否选错了容器

4. 对比时间空间复杂度，选择最优容器
```
